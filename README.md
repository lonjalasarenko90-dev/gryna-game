# 🧠 Гриня рулить

**Технічне обслуговування мозку** — міні-гра про стрес, енергію і те, чому нове життя не починається з понеділка.

![Гриня рулить](https://img.shields.io/badge/React-18-61dafb?logo=react) ![Vite](https://img.shields.io/badge/Vite-5-646cff?logo=vite) ![License](https://img.shields.io/badge/license-MIT-green)

---

## Про гру

Допоможи **Грині** (твоєму мозку) пережити тиждень без перевантаження.

Щодня тобі пропонують 3 картки — корисні дії або стресові події. Твоє завдання: утримати баланс енергії та стресу до кінця рівня.

### Механіка

| Показник | Що означає |
|----------|-----------|
| 🟢 Енергія | Падає від стресу, зростає від відпочинку. Якщо 0 — Гриня здається. |
| 🔴 Стрес | Зростає від тригерів, знижується від турботи. Якщо 100 — паніка. |

### Рівні

| # | Назва | Днів | Складність |
|---|-------|------|-----------|
| 1 | Звичайний понеділок | 5 | 🟢 Легко |
| 2 | Складний тиждень | 7 | 🟡 Середньо |
| 3 | Місяць дедлайнів | 10 | 🔴 Важко |

## Запуск локально

```bash
# Клонуй репозиторій
git clone https://github.com/YOUR_USERNAME/gryna-game.git
cd gryna-game

# Встанови залежності
npm install

# Запусти dev-сервер
npm run dev
```

Відкрий [http://localhost:5173](http://localhost:5173) у браузері.

---

## Збірка для деплою

```bash
npm run build
```

Готові файли будуть у папці `dist/`. Можна деплоїти на:
- **GitHub Pages** — папку `dist` або через GitHub Actions
- **Vercel** — просто підключи репозиторій, команда build: `npm run build`, папка: `dist`
- **Netlify** — аналогічно
